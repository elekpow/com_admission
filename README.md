# com_admission
Joomla com_admission

Вывод списка заявок. 

Версия 0.0.1 

Компонент поддерживает Joomla v5, v6


**возможности в админке:**

Дашборд: Вывод количества заявок, количество опублиокованых, не опублиокованых, и ожидающих решения

Вывод списка заявок. добавление через форму ввода.
